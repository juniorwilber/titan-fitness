# Imágenes locales

Esta carpeta está lista para tus imágenes propias.

Actualmente el sitio usa fotos remotas de Unsplash (ver `index.html`). Para
usar imágenes locales:

1. Coloca tus archivos aquí (por ejemplo `hero.jpg`, `clase-funcional.jpg`).
2. En `index.html`, reemplaza la URL de Unsplash por la ruta local, por ejemplo:
   `src="assets/img/hero.jpg"`.

Recomendación: usa formato `.webp` o `.jpg` optimizado y añade `loading="lazy"`
a las imágenes que no estén en la primera pantalla (ya está aplicado).
