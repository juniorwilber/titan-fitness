# Titan Fitness 💪

Landing page para **Titan Fitness**, un gimnasio urbano ubicado en San Borja, Lima. El sitio presenta las disciplinas del gimnasio (musculación, funcional, spinning, box, HIIT y yoga), los horarios, los coaches y una invitación a probar la primera clase gratis.

**Qué resuelve:** le da al gimnasio una presencia web moderna que muestra su oferta de clases y planes, transmite energía y confianza, y convierte visitantes en nuevos socios captando su primera clase de prueba.

---

## 🧰 Tecnologías

- **HTML5** — estructura y contenido semántico.
- **CSS3** — diseño responsivo, variables y animaciones.
- **JavaScript** — interactividad del lado del cliente.

## 🚀 Cómo verlo localmente

Es un sitio estático: basta con abrir `index.html` en el navegador. Para que todo cargue correctamente, se recomienda un servidor local:

```bash
# Con Python instalado
python -m http.server 8000
# Luego abre http://localhost:8000
```
